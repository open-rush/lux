# @rush/ai-components

AI UI 组件库（React）。用于 apps/web 的聊天界面。

## 规划子模块

```
src/
├── chat/                    # ChatView, MessageList, MessageRenderer
├── tools/                   # ToolRenderer, BashTool, WriteTool, EditTool
├── stream/                  # StreamingText, ReasoningBlock
└── index.ts
```

## 依赖

`@rush/contracts`（UIMessageChunk 类型）, `react`
